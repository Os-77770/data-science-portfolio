## Overview
Developed a cloud-based chatbot using AWS services to automate customer interactions and coffee ordering.

## Technologies
AWS Lex, Lambda, DynamoDB, S3, API Gateway

## Features
- Interactive chatbot interface (hosted on S3)
- Backend processing using AWS Lambda
- Data storage using DynamoDB
- API Gateway integration for communication
- Automated conversation flow for ordering

## Note
This project was built using AWS managed services and console-based configuration, so full source code is not available. The live demo above showcases the system functionality.
