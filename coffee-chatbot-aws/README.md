# Coffee Chatbot (AWS Cloud)

## Overview
Built an AI-powered chatbot using AWS services to automate customer interactions and order processing.

## Technologies
AWS Lex, Lambda, DynamoDB, S3, API Gateway, Python

## Key Work
- Designed conversational chatbot using AWS Lex
- Implemented backend logic using AWS Lambda
- Stored and managed data using DynamoDB
- Built end-to-end pipeline for customer interaction automation

## Result
Successfully automated customer order handling using cloud-based architecture
