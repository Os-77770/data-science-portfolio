## Overview
Built a cloud-based smart door access system that registers visitors, sends OTP codes, and validates access using AWS services.

## Technologies
HTML, JavaScript, AWS API Gateway, Lambda, DynamoDB, S3, SNS/SES

## Features
- Visitor registration page
- OTP validation page
- API Gateway integration
- DynamoDB visitor storage
- OTP delivery through SMS/email
- Cloud-based access verification
